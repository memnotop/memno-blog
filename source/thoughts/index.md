这里可以放一些不一定需要写成长文的记录。短想法可以先在这里沉淀，值得展开的内容再整理成文章。

## 记录方向

<div class="mem-grid">
  <div class="mem-card">
    <h3>阶段复盘</h3>
    <p>一段时间内做了什么、学到什么、哪里需要调整。</p>
  </div>
  <div class="mem-card">
    <h3>读书笔记</h3>
    <p>摘录有价值的观点，结合自己的理解写一点判断。</p>
  </div>
  <div class="mem-card">
    <h3>灵感草稿</h3>
    <p>先记录，不急着完整；后续再筛选、合并和展开。</p>
  </div>
</div>

如果某个想法值得展开，可以新建一篇文章并归类到 `想法`：

```bash
npx hexo new post "文章标题"
```

然后在文章开头设置：

```yaml
categories: 想法
tags:
  - 生活
  - 记录
```
