这里用来存放生活、旅行、设备、截图或其他想长期保存的照片。

建议把照片放在：

```text
source/img/photos/
```

然后在这个页面中用 Markdown 引用：

```markdown
![照片说明](/img/photos/example.jpg)
```

后续可以继续升级成瀑布流相册或独立相册页。
